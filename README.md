# Professional Portfolio - Faisal Akram

A modern, responsive, and minimalist personal portfolio website built with pure HTML and CSS. This project showcases software engineering skills, professional projects, and client testimonials with a sleek developer-focused aesthetic.

## 🚀 Live Preview
Check out the live version here: [Your GitHub Pages Link Here]

## ✨ Key Features
- **Modern Minimalist UI**: Clean white background with professional teal accents (#008080).
- **Infinite Testimonial Slider**: Auto-running slider with manual navigation and infinite loop functionality.
- **Project Showcase**: Interactive project cards with hover effects and Material Icons.
- **Audio Pronunciation**: Custom-built audio button to hear the correct name pronunciation.
- **Simulated Form Submission**: Sleek, split-layout contact form with fake processing states and validation.
- **Developer Aesthetic**: Floating background code symbols and sharp rectangular styling.
- **Fully Responsive**: Optimized for all devices—from desktop to mobile.

## 🛠️ Built With
- **HTML5**: Semantic structure.
- **CSS3**: Custom animations, media queries, and Grid/Flexbox layouts.
- **JavaScript**: Pure JS for slider logic, audio handling, and form simulation (No frameworks).
- **Google Fonts**: Montserrat for modern typography.
- **Material Icons**: For a professional, consistent icon set.
